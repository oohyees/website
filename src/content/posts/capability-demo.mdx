### Counter 计数器

带 `client:only` 的 React 组件，支持状态交互：

<div style="display: flex; gap: 1rem; flex-wrap: wrap;">
  <Counter client:only="react" initial={0} label="点赞数" />
  <Counter client:only="react" initial={42} label="访问量" />
</div>

<Callout type="info" title="关于 React 组件">
`client:load` 和 `client:only` 指令让组件在浏览器端激活。Astro 支持 React、Vue、Svelte、Solid、Preact 等所有主流 UI 框架，同一个项目中可以混用。`client:only` 跳过服务端渲染，适合纯交互组件。
</Callout>

## 4. 可折叠内容

<Collapsible summary="点击展开：更多关于 Astro 的信息">

Astro 是一个现代静态网站生成器，核心特色是：

- **Islands Architecture**：只有必要的交互组件才发送 JS 到客户端
- **零 JS 默认**：纯内容页面完全不加载 JavaScript
- **多框架支持**：同时使用 React、Vue、Svelte 组件
- **Content Collections**：内置类型安全的内容管理系统
- **View Transitions**：原生 MPA 过渡动画

</Collapsible>

```ts file="src/utils/slugify.ts"
export function slugifyStr(str: string): string {
  return str
    .toLowerCase()
    .replace(/\s+/g, "-")
    .replace(/[^\w-]+/g, "");
}
```

## 8. 数学公式

### 行内公式

在物理学中，动能公式 $E_k = \frac{1}{2}mv^2$ 是经典力学的基础。信息论中的熵定义为 $H = -\sum_{i} p_i \log_2 p_i$。

### 块级公式

正态分布概率密度函数：

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}
$$

### 复杂公式

矩阵行列式展开（$3 \times 3$）：

$$
\det\begin{pmatrix}
a & b & c \\
d & e & f \\
g & h & i
\end{pmatrix}
= aei + bfg + cdh - ceg - bdi - afh
$$
